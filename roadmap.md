# Roadmap del proyecto SADEY

## Fase 1 - Fundamentos del proyecto

- Setup inicial con Astro, TypeScript y estructura escalable
- Arquitectura por componentes y datos desacoplados del markup
- Contenido base corregido, depurado y organizado
- Placeholders locales fáciles de reemplazar
- Documentación técnica y editorial
- Configuración de despliegue a GitHub Pages
- Base reusable para futuras iteraciones

## Fase 2 - Diseño visual de la landing

- Aproximación estética a las capturas de referencia
- Mejora del layout general y del ritmo vertical
- Refinamiento del hero, tarjetas, secciones y CTA
- Ajuste fino de colores, tipografía, espaciados y jerarquía visual
- Tratamiento visual más sólido para botones, navbar y footer
- Sin crear subpáginas todavía

## Fase 3 - Reafinado de marca y detalles

- Integración de logo final
- Sustitución de placeholders por imágenes reales
- Revisión fina del copy comercial
- Afinación del branding y consistencia visual
- Microdetalles visuales y de interacción
- Posibles ajustes de navegación antes de escalar a subpáginas

## Fase 4 - Contacto

- Formulario frontend
- Envío a correo
- Validación básica de campos
- Mensajes de éxito y error
- Pruebas funcionales
- Preparación final para dominio propio
